Pisi Linux GitHub setup details

How setup a GitHub for work?

Install at the first Openssh.

sudo pisi it openssh

Next Step install Git.

sudo pisi it git

Create a github ssh-keygen.

ssh-keygen -t rsa -C "youremailadressâ€œ

Open the file ~/.ssh/id_rsa.pub and add the text in this file into the ssh key of your github account.

Create a local global user name.

git config --global user.name "yourgithubnickname"

Make a map for you're github sources in your home, as example 'github' and go to this new map.

Then clone the pisilinux github:

git clone git@github.com:pisilinux/PisiLinux.git

After that clone the playground folder:

git clone git@github.com:pisilinux/playground.git

All the pisilinux buildfile sources are now local, so happy coding.
